zatsu web proxy

Enter a URL to fetch it through this proxy.

Limitations

  • Pages whose body is rendered by JavaScript after load (most SPAs) do not work — only the initial HTML and its referenced CSS / images are rewritten.
  • Cookies and authenticated sessions are not preserved.
  • WebSockets, fetch / XHR from page scripts, and GET forms on the original page may break.
